# Plans Organization

This folder is organized by filesystem creation date using `yyyyMMdd` subfolders.

When you add a new planning artifact, place it in the folder for the day it is created, for example `plans/20260314/`.

Handover files are the exception. Store them in `plans/handover/` and name them using `YYYY-MM-DD_handover.md`.
Use the reusable template at `plans/handover/handover_template.md`.
